Algoritmos BFS y DFS

BFS – Breadth First Search

Explora un grafo por niveles, usando una cola (queue).

Pseudocódigo

BFS(G, inicio):
    crear cola Q
    marcar inicio como visitado
    Q.encolar(inicio)

    mientras Q no esté vacía:
        nodo = Q.desencolar()
        para cada vecino de nodo:
            si no está visitado:
                marcar visitado
                Q.encolar(vecino)

DFS – Depth First Search

Explora en profundidad usando una pila o recursión.

Pseudocódigo

DFS(nodo):
    marcar nodo como visitado
    para cada vecino de nodo:
        si no está visitado:
            DFS(vecino)
