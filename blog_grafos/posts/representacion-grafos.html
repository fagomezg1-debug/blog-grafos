Representación de Grafos

1. Lista de Adyacencia

Ideal para grafos dispersos. Se almacena cada nodo con una lista de sus vecinos.

A → B, D
B → A, C, D
C → B, E
D → A, B, E
E → C, D

2. Matriz de Adyacencia

Una matriz NxN donde 1 indica conexión y 0 ausencia.

    A B C D E
A [ 0 1 0 1 0 ]
B [ 1 0 1 1 0 ]
C [ 0 1 0 0 1 ]
D [ 1 1 0 0 1 ]
E [ 0 0 1 1 0 ]
